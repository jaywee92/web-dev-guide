How CSS Works

There are three ways to add CSS to your HTML pages. They follow a priority cascade:

1. Inline Styles (Highest Priority)
Applied directly to elements via the style attribute
<p style="color: #f472b6;">Styled text</p>
2. Internal Styles
Defined in <style> tags in the document head
<style>
.card { color: #f472b6; }
</style>
3. External Stylesheets (Lowest Priority)
Linked via <link> tags, best for scalable projects
<link rel="stylesheet" href="style.css">

CSS Selectors

Selectors target HTML elements to apply styles. Different selectors have different specificity levels:

Element
p { }
Targets all <p> tags
Class
.card { }
Targets class="card"
ID
#main { }

Descendant
div p { }
All <p> inside <div>
Child
div > p { }
Direct <p> children only
Adjacent Sibling
div + p { }
First <p> after <div>
Pseudo-class
a:hover { }
:first-child, :nth-child(n)
Pseudo-element
p::before { }

CSS in Flask Applications

Flask serves static files (CSS, JS, images) from the /static directory:

File Structure:

project/
├── app.py
├── templates/
│ └── index.html
└── static/
└── css/
└── style.css

In Your Flask Template:

<head>
<link rel="stylesheet" href="{{ url_for('static', filename='css/style.css') }}">
</head>

Using url_for() ensures your CSS path works regardless of deployment environment.
