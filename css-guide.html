🎯 CSS Selectors

📝
Element Selector
Selects HTML elements by tag name
🏷️
Class Selector
Selects elements by .class attribute
#️⃣
ID Selector
Selects element by #id attribute
✨
Pseudo-classes
:hover, :focus, :active, :nth-child()
⚙️
Attribute Selector
Selects elements by attributes
🔗
Combinators
Descendant, child, sibling selectors
Selector Specificity: ID (100) > Class (10) > Element (1). Inline styles (1000) override everything. Use !important rarely.

📦 Box Model

🌫️
Margin
Space outside the element
🟪
Border
Line around the element
📦
Padding
Space inside the element
📄
Content
The actual element content
📐
Width & Height
Element dimensions (px, %, em, etc.)
⚙️
box-sizing
Include/exclude padding and border
Box Model Order: Content → Padding → Border → Margin. Use box-sizing: border-box for easier calculations.

📐 Layout Systems

🔀
Flexbox
1D layout (rows or columns)
📊
CSS Grid
2D layout (rows and columns)
📍
Position
static, relative, absolute, fixed, sticky
👁️
Display
block, inline, inline-block, none
⛵
Float (Legacy)
Old layout method, mostly deprecated
📚
Z-index
Stacking order of elements
Modern CSS: Use Flexbox for 1D layouts, Grid for 2D. Avoid float for layout and use it only for text wrapping.

🎨 Styling Properties

🌈
Colors
text color, background color, gradients
🔤
Typography
font-family, font-size, font-weight, etc.
👥
Shadows & Effects
box-shadow, text-shadow, opacity
🔄
Transform
rotate, scale, translate, skew
⚡
Animation
@keyframes, animation properties
➡️
Transition
Smooth property changes over time
Performance Tip: Prefer transform and opacity for animations. Avoid animating width/height as they trigger layout recalculations.
