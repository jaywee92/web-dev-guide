### IFrames

An `<iframe>` is an HTML document embedded inside another HTML document on a website.

It is a powerful tool for embedding videos, maps, or even interactive content from different sources directly into your web pages.

The `<iframe>` element creates a nested browsing context, where the embedded content is independent of the parent document's content. Attributes like `src`, `width`, `height`, and `style` control the appearance and functionality of iframes.

### Code

The `<code>` tag is used to display a piece of computer code.

It's an inline element that by default, will render the enclosed text in the browser's default monospace font.

This tag is essential for technical documentation or tutorials where showing code examples is necessary.

It's often combined with the `<pre>` tag when displaying multi-line code snippets to preserve formatting and whitespace.

### Forms

Forms are one of the primary methods of gathering input from users on the web.

Through the use of `<form>` elements combined with various types of input elements (like `<input>`, `<textarea>`, and `<button>`), developers can create a wide range of forms for submissions, such as login screens, contact forms, and search bars.

The `action` attribute of the form specifies where the form data should be sent after submission, while the `method` attribute specifies how (GET or POST).

We'll dive deeper into forms very soon. They are, to this day, one of the best ways of interacting with the web, just think of everytime you fill a form to sign in, sign up or submit data!

### Media

Media elements like `<audio>` and `<video>` allow for the embedding of sound and video content directly into web pages.

These elements support several attributes to control playback, such as `controls`, `autoplay`, and `loop`.

Providing an immersive user experience, these tags support various source files, enabling compatibility across different browsers and devices.

It's essential to include the `controls` attribute so users can play, pause, or adjust the volume of the media.
