Logik der Gruppen:
  • Grundlagen — Wie CSS funktioniert, bevor man irgendetwas styled
  • Styling — Text, Hintergründe, Bilder — visuelle Eigenschaften ohne Layout
  • Layout — Wie Elemente angeordnet werden (Flow → Flex → Grid → Responsive)
  • Moderne CSS — Variablen für Wartbarkeit, dann Bewegung/Effekte aufbauend
