Flexbox

CSS Layout Level 2

Eindimensionales Layout-System für flexible Elemente in Reihe oder Spalte. justify-content, align-items, gap, flex-wrap.

Visualisierung
A
B
C
display: flex justify-content: center
Erklärung

Mit display: flex wird ein Container zum Flex-Container. Alle direkten Kinder werden zu Flex-Items. justify-content steuert die Hauptachse.

.container {
display: flex;
justify-content: center;
gap: 16px;
}
